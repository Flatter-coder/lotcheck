# ✅ LotCheck

**Did you LotCheck it?**

Canada's car market intelligence platform.

## What it does
- 📈 Tracks used car prices over time like a stock chart
- ✅ LotScore — rates every deal vs the market
- ⚡ EVAP rebate checker — federal + provincial stacked
- 📊 Canadian Black Book estimates (Pro)
- 🗓️ Alberta Allocations Tracker (Pro)
- 🤝 Connects buyers to dealer partners

## Tech stack
- React + Vite
- Recharts (price charts)
- Supabase (database)
- Apify (scraper)
- Vercel (hosting)
- Stripe (payments)
